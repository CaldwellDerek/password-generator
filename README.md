# Password-Generator

## Description
The motivation behind this project was to demonstrate understanding of Javascript and how it can be used to interact with HTML / CSS via the DOM. Here, the HTML button 'Generate Password' responds to an event listener and displays a series of prompts to determine a suitable password based off of the user's chosen criteria.

## Installation
N/A

## Usage
Clicking on 'Generate Password' will begin the process. After which, a series of prompts will display giving the user options as to what criteria their password must meet. After all prompts are answered, the generated password is displayed in the textbox.

This project can be viewed at the following:

https://caldwellderek.github.io/password-generator/
